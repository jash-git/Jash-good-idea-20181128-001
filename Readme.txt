尿完多看兩眼！這四種尿液顏色，代表健康出問題


資料來源:https://www.setn.com/News.aspx?NewsID=461819


1.黃色：淺黃、深黃而透明的尿液都屬於正常範圍，差異在於飲水量多寡，喝的水多，則尿色較淡。
 
2.橘色：若尿色呈現橘黃色，表示身體已經開始缺水囉，建議應盡快補充水分。
 
3.褐色：身體嚴重缺水，甚至可能呈現脫水的現象；另外，褐色尿液亦可能為肝臟疾病所致。

4.紅色：尿液帶有紅色（深紅、淺紅、粉紅皆可能出現）可能是血尿的症狀，恐為泌尿道或腎臟疾病所致。然而，若曾大量攝取甜菜根、紅色火龍果等食材，亦可能出現紅色尿液，建議可稍加留意觀察，血尿症狀是否持續出現。

5.藍色或綠色：尿液出現罕見的藍色或綠色可能代表有遺傳性疾病高血鈣症，或是受到細菌感染，建議盡快就診檢查。
